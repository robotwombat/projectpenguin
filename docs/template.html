These are some steps.
For flashing the fastboot zip
MAKE SURE YOU ARE ON LATEST VERSION OF PLATFORM TOOLS!
1. Unlock bootloader
2. Download the fastboot image of the rom (the zip file with "-img" in it)
3. Boot to fastboot
4. Type
fastboot update location_of_zip_file
(DO NOT use fastboot -w to wipe data, follow next step)
5. Boot to inbuilt recovery if you are flashing for the first time and format data. (F2FS is preferred)
6. Reboot.
